# AI 活用上のガイドライン

出典: 業務リスト Excel「AI活用効果サマリー」シートの注意点を作業用に展開。

## 推奨ツール

- Claude / ChatGPT: 翻訳・要約・論点抽出・比較表初稿
- DeepL: フランス語翻訳補助
- **ヒアリングの非公開情報は AI に投入しない**

## ブロック別の削減率の目安

| ブロック | AI の主用途 | 人間の役割 |
|----------|-------------|------------|
| (1) 政策・制度 | フランス語文献の翻訳・要約 | 原文照合・制度解釈 |
| (2) 技術・手法 | 概念整理・比較表初稿 | 専門的精査・算定根拠 |
| (3) 情報基盤 | DB 構造・パラメータ整理 | 仕様の正確性確認 |
| (4) ヒアリング | 質問票・議事録要約 | 実施・関係構築 |

## 必須ルール

1. **ハルシネーション対策**: すべての AI 出力をフランス語・英語原文と照合する
2. **対訳辞書**: `glossary.md` を参照させ、用語を統一する
3. **機密**: 非公開のヒアリング内容はローカル限定・AI 非使用
4. **納品物**: AI 出力をそのまま提出しない — 必ず編集・統合する
5. **引用**: 出典（文献名・ページ・URL・日付）をメモに残す

## 文献メモでの記載例

```
【AI使用】要約初稿: Claude, 2026-05-XX
【検証】guide_re2020 p.XX と照合 — 担当: 氏名
```
