  2. CERIB反対論拠の詳細化: v3では「コンクリート産業が反対した」という事実のみ記載。v4では反対論拠4点（EN整合・貯蔵の無意味性・SCM評価・ツール対応）と政府反論のセットが提示され、「どのように合意形成が進んだか」が理解できる。
