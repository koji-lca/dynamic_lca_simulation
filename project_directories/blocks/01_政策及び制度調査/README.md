# 第1係：政策及び制度調査

**調査ブロック (1)** — フランス RE2020 における建築物 LCA・ダイナミック LCA の制度設計と政策背景

## 係のミッション

RE2020 を中心に、動的 LCA・生物由来炭素・炭素貯蔵評価が**制度上どう位置づけられているか**を整理し、日本の建築物 LCA 制度化に向けた政策・制度面の示唆を抽出する。

## 他係への引き渡し

- **第2係**: 制度上の「遅延排出」「評価期間」等の要件 → 技術的解釈の前提
- **第3係**: 貯蔵量算定・EoL 扱い → INIES・炭素フローモデルとの対応確認
- **第4係**: 制度運用上の論点・ヒアリング質問のたたき台

- RE2020 と建築物 LCA の全体像
