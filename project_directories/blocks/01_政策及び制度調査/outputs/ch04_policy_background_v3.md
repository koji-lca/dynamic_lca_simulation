## エグゼクティブサマリー：4つの収束した必然性

フランスがRE2020で動的LCAを世界に先駆けて規制化した理由は、単なる「先進的政策への意欲」ではない。以下の**4つの構造的必然性が同時に成立**したことが決定要因であった。

> **核心命題:** 静的LCA（GWP100）は、木材建築物の炭素貯蔵という**実在する気候便益を計測不能な評価指標**であった。フランスはこの測定ツールの失敗を修正することで、気候目標・産業戦略・国際標準化の三目標を同時に達成した。

| 必然性 | 内容 | 収束点 |
|--------|------|--------|
| **① 科学的必要性** | 静的GWP100は「後の排出ほど気候影響が小さい」という物理的事実を無視する構造的欠陥を持つ | 評価ツールの失敗が政策の失敗を引き起こしていた |
| **② 政策的整合** | SNBC 2050目標の達成には、建材選択への正確な炭素シグナルが不可欠 | 静的LCAの修正なしにはSNBC目標は達成不可能 |
| **③ 産業戦略的利益** | フランスの木材産業は静的LCAによって「公平な競争条件」を与えられていなかった | 測定ツールの修正＝産業政策目標の達成 |
| **④ 国際先行者戦略** | EU規格・国際規格の議論を先行する自国制度で主導し、国際競争上の優位を確保 | EPBD 2024への先行モデル提供・CEN/TC 350規格改訂議論への影響力として現れる |

> **⚠️ 日本の判断にあたっての留意（本章の中立的立場）:** 上記4点はフランスが動的LCAを選んだ「理由」の分析であり、**日本も同じ選択をすべきという結論ではない**。動的LCAを採用する国はフランスのみで、EU・北欧の主流は静的LCAである（§7.3）。一時的炭素貯蔵に便益を与えること自体への科学的異論（§9.2）、データ基盤・コストの課題も存在する。導入の是非・代替手段・段階的経路は §9 で中立的に整理する。本報告書は導入を推奨も否定もしない。

## 1. 静的GWP100の構造的失敗——なぜ既存ツールでは不十分だったか

> **本節で分かること:** フランスが静的LCAを「使えない」と判断した科学的根拠。専門家委員会・省庁関係者がこの判断を理解するために不可欠な基礎論証。

### 1.1 GWP100の前提と建築物への適用限界

GWP100（地球温暖化係数、100年時間地平）は、ある物質1kgを大気に放出したときの累積放射強制力（CRF: Cumulative Radiative Forcing）を基準物質（CO₂）のそれと比較した無次元指数である（IPCC, 2007）。

**GWP100の前提（製品評価では妥当）:**
- 評価の「時間原点（t=0）」は排出時点で固定
- 異なる製品・工程の排出を同一時点の等価量として比較する
- 製品のライフサイクル内で排出と吸収のタイミングはほぼ等しいと仮定

**建築物ライフサイクル評価への適用で生じる問題:**  
建物には明確な「吸収時点」と「排出時点」の**時間的非対称性**が存在する。木材建材の場合：

```
t = 0年（建設時）: 木材が大気から固定したCO₂を建物として封存  →  吸収として計上
t = 50年（解体時）: 封存していたCO₂が解体・焼却で大気に放出  →  排出として計上
```

静的GWP100はこの50年の差を無視し、「吸収1kgCO₂ + 排出1kgCO₂ = 正味ゼロ」と評価する。  
**すなわち、静的LCAにおいて木材はコンクリートと等しく「炭素的に中立」と評価される。**

しかし、これは物理的事実と一致しない。

### 1.2 物理的現実：時間的変位が気候影響を変える

50年後に放出された1kgCO₂は、今日放出した1kgCO₂と同じ気候影響をもたらさない。理由はIPCC AR4 Bern2.5CCモデルで定量化されている：

$$DCF(t_i, DO) = \frac{CRF(t_i, DO)}{CRF(0, DO)}$$

ここで $CRF(t_i, DO)$ は時刻 $t_i$ に排出した1kgCO₂が与える累積放射強制力であり、これは将来の海洋吸収・生物圏吸収が寄与するため、$t_i$ が大きいほど小さくなる。

| 排出時点 $t_i$ | RE2020重み係数（$1-0.00842t$） | 気候影響の重み |
|------------|---------------------|------------|
| t = 0年（今日） | 1.000 | 基準（100%） |
| t = 25年 | 0.790 | 約79% |
| t = 50年（設計寿命） | 0.578 | 約58%（＝**42%の影響減**） |

（RE2020公式係数は排出年の一次関数 $1-0.00842t$、t=50で0.578〔Arrêté 2021第11条; Cerema, 2022〕。なお§5.2で述べる通り、この58%という重みは「正しい値より木材に有利な方向に約25〜35%の過大評価」を含むとの指摘がある〔Ventura, 2022; Univ Gustave Eiffel〕）

### 1.3 測定ツールの失敗が設計シグナルの失敗を引き起こす

木材建材（CLT）を静的LCAと動的LCAで評価すると：

```
【静的LCA（GWP100）】  
  A1-A3（建設、t=0）: バイオジェニック炭素固定 = −1.0 kgCO₂/kgC  
  C4（解体、t=50年）: バイオジェニック炭素放出 = +1.0 kgCO₂/kgC  
  ─────────────────────────────────────────────────────────────────  
  正味合計 = 0.0 kgCO₂/kgC  ← 木材 ≡ コンクリート（どちらも "中立"）

【動的LCA（RE2020）】  
  A1-A3（建設、t=0）: DCF(0) = 1.0  → 計上 −1.0 kgCO₂/kgC  
  C4（解体、t=50年）: DCF(50) = 0.58 → 計上 +0.58 kgCO₂/kgC  
  ─────────────────────────────────────────────────────────────────  
  正味合計 = −0.42 kgCO₂/kgC  ← 木材はコンクリートより 0.42単位 有利
```

これを建材レベルで見ると（第1編報告書 §4.7）：

| 建材 | 静的LCA [kgCO₂eq/m³] | 動的LCA [kgCO₂eq/m³] | 評価の変化 |
|------|---------------------|---------------------|----------|
| **CLT（直交集成板）** | +98（排出） | **−171（吸収）** | **−269の逆転** |
| **製材** | +65（排出） | **−115（吸収）** | 逆転 |
| RC造コンクリート | +300前後 | +295前後 | **ほぼ不変** |
| 構造用鋼材 | +550前後 | +548前後 | **ほぼ不変** |

**政策的含意:** 静的LCAを使い続ける限り、設計者は木材を選んでもコンクリートを選んでも「同じ」という誤ったシグナルを受け取る。動的LCAへの移行は、実在する気候便益を設計決定に正確に反映させるための**測定ツールの修正**である。

## 2. なぜ今（2022年）だったのか——4条件の収束

> **本節で分かること:** 「動的LCA規制化は理論的には2009年時点で可能だったが、なぜ2022年になって初めて実現したか」という制度的・実務的条件の分析。

### 2.1 規制化の4条件

| 条件 | 必要な状態 | フランスの達成時期 |
|------|-----------|-----------------|
| **科学的基盤** | 査読を経た動的LCA手法の確立 | 2009年（Benoist）・2010年（Levasseur）の研究を基礎に2010年代後半に実務化 |
| **データ基盤** | 十分なFDES/EPDカバレッジ | E+C-プログラム（2016-2019）で整備が進む |
| **計算ツール** | 実務者が使えるソフトウェア | Elodie（CSTB）が動的LCA計算を自動化（2018〜） |
| **政治的正当性** | 産業界・規制当局の合意 | E+C-実証データ＋FCBA支持＋EUグリーンディール整合 |

**4条件が全て揃ったのが2021〜2022年。これが「今」の理由である。**

### 2.2 E+C-プログラム：証拠に基づく規制化の先例

2016〜2021年の実証プログラム「E+C-（Énergie Positive et Réduction Carbone）」は、RE2020にとって単なる「予備的試行」ではなく、**規制設計のエビデンスベースを構築する計画的プロセス**であった。

- **参加規模:** 公開観測データベースに約830事業が登録（住宅中心、第三次産業含む。2016〜2021年累計）
- **得られたデータ:** 建物種別・気候ゾーン別の実際の $Ic$ 値の分布  
  → これがフェーズ1 seuil値（住宅：640 kgCO₂eq/m²等）の根拠
- **明らかになった課題:** FDESデータのカバレッジ不足、計算ツールの習熟コスト
- **産業界の反応:** 参加企業から「基準値の設定根拠が透明」との評価

E+C-がなければ、seuil値は根拠なき行政的数値に過ぎなかった。フランスはこの「10年の準備」を省略しなかった。

## 4. なぜ他の手段ではなく動的LCAだったのか——代替案との比較

> **本節で分かること:** 「動的LCAの代わりに使えた他の政策手段が検討されたが採用されなかった理由」を明示することで、動的LCAの選択が消去法でなく積極的選択であることを示す。

### 4.1 検討された代替手段と却下の理由

| 代替手段 | 概念 | 採用されなかった理由 |
|---------|------|-------------------|
| **静的LCA（GWP100）のまま使用** | 現行手法を継続 | §1で述べた通り、木材の気候便益が計測不能。設計シグナルが誤り。SNBC目標達成に不十分と判断 |
| **木材への固定補助金・クレジット制度** | 木材建材を使う建物に一律炭素クレジットを付与 | 物理的根拠のない固定値では科学的正当性なし。補助金は財政負担。EU規制との整合困難 |
| **静的LCA + バイオジェニック炭素の別途報告（EN 15804準拠）** | −1/+1アプローチのみを採用し、時間補正は行わない | 「吸収t=0・放出t=50年」の物理的非対称性が評価に反映されず、§1.1の問題が残存 |
| **完全な動的LCA（非線形DCF・可変時間地平方式）** | Ventura (2022) が提案する TOD = LCD + THI 方式 | 設計者・計算ツールへの要求が高く、2022年段階では実装不可能。過渡期的に線形近似を採用し、2028年フェーズ3での見直しを計画 |
| **建築炭素税** | 建材の炭素含有量に応じた税 | 産業界の反発が強く政治的に困難。炭素価格の変動リスク。EU ETSとの重複問題 |

**結論:** 動的LCA（線形近似）は、科学的妥当性・実務的実現可能性・政治的受容性の三要件を唯一満たす手段であった。

### 4.2 線形近似の科学的トレードオフ

フランスが採用した線形近似は、厳密な非線形DCFよりも不正確である（Ventura, 2022）。しかしこの選択には明確な政策的合理性がある：

| 評価軸 | 非線形DCF（可変DO方式） | RE2020線形近似 |
|--------|---------------------------|--------------|
| **科学的精度** | ◎（物理的に正確） | △（t=50年で木材便益を約25〜35%過大評価） |
| **計算の透明性** | △（数値積分が必要） | ◎（係数表で誰でも再現可能） |
| **実務コスト** | 高（専門的ソフトウェア必須） | 低（Excelでも計算可能） |
| **規制の予測可能性** | 低（DOの定義に解釈余地） | 高（パラメータ完全固定） |
| **2022年時点での実装可能性** | △ | ◎ |

フランスは**2022年の規制化を優先し、2028年フェーズ3での精度向上を計画的に留保した**。これはアダプティブ・レギュレーション（adaptive regulation）の意図的設計である。

## 5. 政策目標・課題認識と批判的考察

### 5.1 RE2020が達成した5つの目標

1. **建築エンボディド炭素の「見える化」（主目的）:**  
   RT2012では建材・施工起因のGHGが評価外。断熱性能を上げるために大量の断熱材（高炭素）を使うというトレードオフが見えなかった。$Ic_{\text{construction}}$ 規制がこの盲点を解消した。

2. **設計段階の炭素シグナルの修正（科学的目標）:**  
   §1.3で示した「静的LCA：木材≡コンクリート」という誤ったシグナルを修正し、木造建築の正当な評価を可能にした。

3. **段階的目標による継続的技術革新の制度化（政策工学目標）:**  
   フェーズ1→4の段階的seuil引き下げは、毎フェーズで建設業界全体に低炭素化のプレッシャーをかけ、FDES整備・設計ノウハウ蓄積・材料革新を促すメカニズムとして機能する。

4. **産業競争条件の再調整（産業政策目標）:**  
   木材産業にとって不利だった「静的LCAの土俵」を動的LCAに変えることで、木造建築の経済的競争力を正当化。**フランス政府はこれを「補助金」ではなく「正確な計測への移行」として正当化した**。

5. **EU規格の先取り（国際戦略目標）:**  
   EPBD 2024（WLC開示義務）の手法論としてRE2020方式が参照されることを見越した先行実装。CEN/TC 350における欧州規格改訂の議論への影響力確保とも連動している。

### 5.2 残された構造的限界（専門家委員会への正直な提示）

専門家委員会に対しては、現行制度の限界も隠さず提示する必要がある：

| 課題 | 内容 | 深刻度 | 2028年以降の対処 |
|------|------|--------|----------------|
| **DO固定による過大評価** | 一時的炭素貯蔵の効果がt=50年時点で約25%過大評価される（Ventura, 2022。個別係数値は原典照合中） | 中（方向性は正しい） | 非線形DCF（可変時間地平）方式への移行を検討 |
| **FDES品質・カバレッジ不足** | デフォルト値（過度に不利な値）に依存する建材が多い。設計者間のIc値ばらつきが大きい。2025年にデフォルト値モデュレーション（Mided）が廃止され、FDES未整備の建材はさらに不利に | 高 | FDES整備の加速（Rivaton報告書〔2025・23提言〕を受けた政府の調整プロセスで検討。個別提言番号は原文照合中） |
| **中小業者コスト負担** | EPD取得・計算費用が中小建設業者に集中 | 中 | 公的支援・一括ツールの提供 |
| **Dモジュール除外** | 再利用・リサイクルの炭素便益がIcに反映されず、サーキュラー設計への誘因が弱い | 高 | 部分的Ic組み込みの検討（個別提言の有無は原文照合中） |
| **SFM前提の脆弱性** | 「伐採＝植林」が保証されない場合の評価方法が未確立 | 高（長期的） | FSC/PEFC認証との連動強化 |

## 6. フランスの林業・木材利用戦略との関係性

### 6.1 なぜ木材産業がこれほど強く動的LCAを支持したか

FCBAを中心とする木材産業界が動的LCAを積極的に支持した経済的論理：

```
【静的LCAの世界（2022年以前）】
  コンクリート建物: Ic ≈ +300 kgCO₂eq/m³
  木材（CLT）建物: Ic ≈ +98 kgCO₂eq/m³（低いが「排出」のまま）
  → 設計者へのメッセージ: 「木造の方が少し良いが、根本は同じ（どちらも排出）」

【動的LCAの世界（RE2020 2022年〜）】
  コンクリート建物: Ic ≈ +295 kgCO₂eq/m³（ほぼ不変）
  木材（CLT）建物: Ic ≈ −171 kgCO₂eq/m³（「吸収」に逆転）
  → 設計者へのメッセージ: 「木造を選ぶことは積極的炭素固定を意味する」
```

この評価の逆転（「排出」→「吸収」）は、設計者・建主の木造選択率に直接影響する。フランスの木材産業はこれを「市場アクセス拡大」ではなく「正当な評価の回復」として訴えた（第1編報告書 §2.3）。

### 6.2 動的LCAと林業政策の双方向強化ループ

```
RE2020（動的LCA）が木造建築を優遇
        ↓
木造建築需要の拡大 → 国産木材消費量の増加
        ↓
FSC/PEFC認証林の面積拡大・持続可能な森林管理の強化
        ↓
森林炭素ストックの信頼性基盤が向上（StockC指標・動的評価の前提が強化）
        ↓
動的LCAの科学的前提（SFM維持）が強化される
        ↑（ループ）
```

この正のフィードバックループが、フランスにおいて気候政策・林業政策・建設産業政策の三者を連動させる制度的基盤となっている（Rivaton, 2025; 第1編報告書 §2.3）。

### 6.3 主要な林業・木材政策との具体的連動

| 政策 | RE2020との接続ポイント | 定量的効果 |
|------|---------------------|----------|
| **France Bois 2024**（国産木材利用計画） | RE2020のseuil引き下げが木造設計の採用を促進 | 木造建築着工比率の向上（具体数値要確認） |
| **気候レジリエンス法（loi n°2021-1104）第39条** | 公共発注の新築・大規模改修に**生物由来材等25%以上**の使用を義務化（2030年1月施行予定）。RE2020のIc規制と組み合わせ木材利用を促進 | 公共建築での木材・生物由来材比率の底上げ |
| **France 2030**（木材産業関連投資） | 木材産業への投資（投資規模・対象は要確認）がFDES整備・EPD作成を支援 | FDESカバレッジの拡大 |
| **PEFC/FSC認証拡大** | StockCのSFM前提をFDESデータで証明するための基盤 | バイオジェニック炭素クレジットの信頼性向上 |

## 7. 国際標準先行者戦略——フランスの地政学的計算

### 7.1 なぜ「国際標準を先に形成する」戦略が重要か

国際標準は、それを先に策定した国が「自国の手法を世界標準として書き込む」機会である。フランスは以下の戦略的計算を持っていた：

1. **欧州・国際標準化議論への主導参加:** AFNOR（フランス規格協会）を通じ、CEN/TC 350におけるEN 15978/EN 15804改訂等の標準化議論に参加。動的バイオジェニック炭素評価を直接規定する国際規格は未成立だが、RE2020の規制運用実績がフランスの発言力の基盤となっている。

2. **EPBD 2024への影響力:** EU建築物エネルギー性能指令の2024年改正でWLC評価義務化が決まったが、方法論としてRE2020が「先行事例」として参照される。フランスが先に制度を持つことで、EU規格策定において自国方式を擁護しやすくなった。

3. **EU議長国（2022年上半期）の活用:** RE2020施行と同時期にEU議長国となり、欧州建築環境政策の議題設定を主導した。

### 7.2 国際展開の成果と限界

| 成果 | 内容 |
|------|------|
| 標準化議論への影響力 | 動的LCAの規制運用実績を持つ唯一の国として、CEN/TC 350等の規格改訂議論に先行知見を提供 |
| EPBD 2024参照事例 | EU全域WLC義務化の方法論的先例として位置づけ |
| EU建築部門炭素規制の先行実装 | スウェーデン・ドイツ等が静的LCA段階の中、フランスが3年先行 |

| 限界・課題 | 内容 |
|-----------|------|
| EN 15804との哲学的相違 | EUの建材EPD規格（EN 15804）は静的LCAベースのまま。RE2020との手法整合が産業界に混乱 |
| 非線形DCFとの乖離 | 学術的にはRE2020より精密な非線形DCF（可変DO）が提案されており、将来規格化された場合はフランス国内制度の「アップグレード」が必要になる可能性 |

### 7.3 各国の建築物炭素規制の比較——「動的LCAはフランスだけ」という重要事実

日本の制度設計判断にあたり最も重要な事実は、**ライフサイクル炭素を「国家規制」として義務化した国は複数あるが、その中で動的LCA（排出タイミングの重み付け）を採用しているのはフランスのみ**である点である。先行する規制国はすべて静的LCAを採用している。一方、**規制ではない自主基準・研究レベルでは、動的LCAは複数国で採用が進んでいる**（ノルウェー・米国等）。この「規制か／自主・研究か」の区別が判断上きわめて重要である。

#### (a) 国家規制（義務）レベル

| 国 | 制度（開始年） | 手法 | 規制値の例 | 特徴 |
|----|--------------|------|----------|------|
| **オランダ** | MPG（2013年義務化、2018年に上限値導入） | 静的LCA | 環境コスト指標（€/m²/年）。住宅0.8等 | エンボディド環境性能を最初に義務化した国。炭素単独でなく統合環境指標 |
| **フランス** | RE2020（2022年） | **動的LCA（簡易）** | Ic construction 640〜980 kgCO₂eq/m²（用途別・50年） | **唯一の動的LCAを義務化した国**。木材の時間貯蔵を制度評価 |
| **スウェーデン** | Klimatdeklaration（2022年） | 静的LCA | 当初は申告義務のみ（上限値は2027年予定） | まず「測る・申告する」から開始する段階的アプローチ |
| **デンマーク** | Bygningsreglementet/BR18改正（2023年） | 静的LCA | 全用途 7.1 kgCO₂eq/m²/年（2025年7月〜。2023年は>1,000m²に12 kg） | 年あたり kgCO₂eq/m²/年 で表現。隔年で厳格化 |
| **フィンランド** | 建築法改正（2025年〜） | 静的LCA | 用途別上限値 | 北欧の同期的導入の一環 |
| **EU全体** | EPBD改正（2024/1275） | 手法はCEN委任（静的GWP基盤） | 2028年:大規模建物、2030年:全新築でWLC開示義務 | 上限規制は各国裁量。手法詳細はEN改訂待ち |

#### (b) 自主基準・研究・ツール開発レベル（規制ではない）

| 主体 | 取組（開始年） | 手法 | 性格 | 特徴 |
|------|--------------|------|------|------|
| **ノルウェー** | **FutureBuilt ZERO**（2022年、SINTEF開発） | **動的LCA（簡易、時間依存特性化係数）** | オスロ圏の**自主パイロット基準**（規制ではない） | 評価期間60年。基準建物比**GHG50%削減**を要件化。生物由来炭素・セメント炭酸化・再利用可能性も算入。RE2020と同じ「簡易動的」系統だが要件設計は独自 |
| **米国** | **POD\|LCA**（Parametric Open Data for LCA、2023年〜） | **動的LCA・time-explicit LCA**を枠組みに包含 | Carbon Leadership Forum＋ワシントン大の**研究・ツール開発**（約$3.7M／4年）。規制ではない | 生物由来炭素・動的LCA・炭素貯蔵を全ライフサイクルで扱う比較LCAフレームワーク。米国に建築炭素の国家規制はなく、州・自治体のBuy Clean等は静的EPDベース |

> **判断材料としての含意:**
> 1. **「規制」の主流は静的LCA、「自主・研究」では動的LCAが拡大中**という二層構造がある。義務規制で動的LCAを採るのはフランスのみだが、ノルウェーFutureBuilt ZERO（自主）・米国POD\|LCA（研究）のように、より柔軟な枠組みでは動的LCAが採用されつつある。**日本が動的LCAを検討する場合、いきなり規制化するフランス型より、まず自主基準・実証ツールで導入するノルウェー/米国型の方が、現状（データ基盤未整備）に適合し得る。**
> 2. FutureBuilt ZEROもRE2020と同じ「簡易動的（時間依存特性化係数）」系統だが、評価期間（60年 vs 50年）・要件（相対50%削減 vs 絶対seuil）が異なる。**動的LCAにも複数の設計流派があり、「動的LCA＝RE2020方式」ではない。**
> 3. デンマークの「kgCO₂eq/m²/**年**」とフランスの「kgCO₂eq/m²（50年）」のように、**規制値の単位・正規化方法すら国際的に未統一**である。動的か静的か以前の、より基礎的な制度設計論点が国際的に未収束であることを示す。
> 4. 「測る・申告する」から始めて数年後に上限値を導入したスウェーデン方式、自主基準から入るノルウェー方式は、いずれもデータ基盤が未整備な国にとって現実的な移行経路の一例である。

| フランスの成功要因 | 日本の現状 | 転用可能性 |
|-----------------|----------|----------|
| **E+C-型の実証プログラム** | CASBEE等の任意認証はあるが動的LCA実証は未実施 | **高い**。CASBEE改訂とセットで実証実験設計が可能 |

| **FCBAに相当する産業的支持** | CLT普及協会・日集協等が潜在的支持勢力 | **高い**。動的LCAが日本の国産CLTに与える評価逆転効果はフランスと同等 |
| **国家戦略との整合** | 林野庁の木材利用促進法・建築基準法改正 | **高い**。気候・林業・建設の三政策の収束点として動的LCAを位置づけ可能 |

### 8.2 日本版「なぜ今か」の論証

日本が動的LCA導入を検討する場合、フランスと同様の4条件を以下のように整理できる：

```
【日本版4条件の現状（2026年時点）】

① 科学的基盤:   DCFの方法論は査読文献で確立（Levasseur 2010等）。ただし国際規格は未成立であり、
                国内ガイドラインは学術的基盤の上に独自設計が必要（標準化動向の注視が前提）
② データ基盤:   IDEAへの時系列炭素フローデータ追加が急務（現在未整備）
③ 計算ツール:   動的LCA対応ツールが国内に存在しない（最優先開発課題）
④ 政治的正当性: 木材利用促進基本計画・カーボンニュートラル宣言と整合するが
                産業界の合意形成が未形成

→ ②③④が未整備。「実証なき規制化はしない」フランスの教訓に倣い、
  5年間の実証プログラム設計が最初のステップ。
```

### 8.3 日本が避けるべきフランスの失敗

1. **DOの固定問題の再現:** フランスはDO=100年固定で過大評価問題を抱えた。日本は非線形DCF（変動DO）の採用可能性を最初から検討することで、この失敗を回避できる。
2. **データ後追い:** フランスはFDESカバレッジ不足のまま規制化し、デフォルト値依存が慢性化した。日本は規制化前にIDEA/日本版FDESの時系列データ整備を完了させるべき。
3. **中小業者コスト無視:** フランスの失敗事例を参考に、日本では計算ツールの無償提供・行政コスト補助を制度設計に織り込むべき。

## 9. 動的LCA導入の是非——官僚的判断のための論点整理

> **本節の位置づけ:** 本報告書は調査項目①の制度分析であり、導入の是非そのものを結論づけるものではない。しかし林野庁の最終判断（導入すべきか、すべきなら何に注意すべきか）に資するため、判断の分岐点と論拠を中立的に整理する。

### 9.1 動的LCAを支持する論拠

1. **物理的正確性:** 静的GWP100は排出タイミングを無視し、木材の数十年の炭素貯蔵という実在する気候便益を計測できない。動的LCAはこの測定上の欠陥を修正する（§1）。
2. **政策整合性:** 木材利用促進基本計画・2050カーボンニュートラルと、木材を正当に評価する動的LCAは方向が一致する。
3. **産業競争条件の是正:** 静的評価下で「排出材」とされる国産CLT等が、動的評価で「固定材」に逆転し得る（§6）。
4. **先行者利益:** 国際標準化（CEN/TC 350等）が未収束の今、実証知見を持つことで標準形成に関与できる。

### 9.2 動的LCAに慎重・反対すべき論拠（同等に重要）

1. **国際的孤立リスク:** 動的LCAを採用する国はフランスのみ。EU・北欧・EN 15804は全て静的LCA基盤（§7.3）。動的を選ぶと国際的なEPD・データ互換性で不利になり得る。
2. **科学的論争が未決着:** 一時的炭素貯蔵に「気候便益」を与えること自体に学術的異論がある。Brandão et al. (2013) 等は、貯蔵は排出を遅らせるだけで大気中濃度の最終的な積分値を減らさない可能性を指摘し、貯蔵クレジットの正当性を問う。RE2020の手法もVentura (2022) に約25%の過大評価を指摘されている。**「動的LCAが静的より正しい」は自明ではない。**
3. **制度の複雑性とコスト:** 動的LCAは計算ツール・データ基盤・実務者教育への要求が高い。フランスはFDESカバレッジ不足とデフォルト値依存に苦しみ、2025年のMided廃止で中小事業者負担が増した。+11%の投資コスト増（2035年見通し、Rivaton 2025）も報告されている。
4. **データ前提の脆弱性:** 木材の便益は「伐採＝再植林（SFM）」を暗黙の前提とする。国産材でこの前提が崩れる地域があれば、便益計上の正当性自体が揺らぐ（§3.4）。
5. **目的に対する過剰性:** 木材利用促進が主目的なら、補助金・公共調達義務（仏の生物由来材25%義務等）・容積率優遇など、より直接的で制度コストの低い手段が存在する。動的LCAは「正確な計測」のための手段であり、「木材振興」の最短経路とは限らない。

### 9.3 導入オプションの4類型

是非は二択ではない。少なくとも4つの選択肢がある。

| オプション | 内容 | 向くケース |
|----------|------|----------|
| **A. 動的LCAを規制化** | フランスRE2020型。排出タイミングを重み付けし義務化 | 木材の時間便益を制度評価したい／実証データ基盤を整備できる |
| **A'. 動的LCAを自主基準・実証から導入** | ノルウェーFutureBuilt ZERO／米国POD\|LCA型。規制化せず自主基準・ツールで先行 | 動的LCAに関心はあるがデータ・合意が未成熟（**日本の現状に適合的**） |
| **B. 静的LCA＋炭素貯蔵の別途開示** | 北欧規制型。規制は静的GWP100、貯蔵量は情報提供（フランスのStockC指標に類似） | 国際整合を優先しつつ木材の貯蔵を可視化したい |
| **C. 申告義務のみ先行** | スウェーデン型。まず測定・申告を義務化し、上限値は数年後 | データ基盤が未整備（＝日本の現状に最も近い） |
| **D. 非規制的手段で代替** | 補助金・公共調達・税制等で木材利用を直接促進 | LCA制度化の合意・コストが見合わない |

### 9.4 「導入しない／急がない」が合理的となる条件

本報告書は導入を推奨も否定もしない。ただし以下の条件が揃う場合、**「現時点では導入しない、またはオプションC/Dを選ぶ」ことが合理的**である：

- 建材環境データベース（IDEA等）に時系列炭素フロー情報がなく、整備に数年を要する → 動的LCAは計算不能
- 動的LCAの科学的論争（貯蔵クレジットの正当性、DO固定の過大評価）が未決着であり、官僚が説明責任を負えない
- 木材利用促進が主目的なら、より直接的な政策手段（オプションD）の費用対効果が動的LCA導入を上回る
- 国際標準（CEN/TC 350・EPBD下のEN改訂）の方向性が固まるまで待つことで、手戻りリスクを回避できる

逆に、**実証プログラムで国内データを蓄積しつつ（オプションC）、国際標準化の議論に参画し、その上で動的（A）か静的＋開示（B）かを判断する**という段階的経路が、現実的な落としどころとなり得る。
