## エグゼクティブサマリー

**3つのKey Finding:**
1. **No.21（算定環境）:** RE2020のLCA算定エコシステムは「INIES/FDESデータベース → Elodie計算ソフト → 申請機関への提出」という三層構造で成立する。フランスはこの計算環境を無償公開しており、導入障壁は他国比で低い。一方、FDESカバレッジ不足とデフォルト値依存が最大の実務課題である。
2. **No.22（静的LCA関係性）:** RE2020は「静的LCA＋時間補正係数」という**ハイブリッド設計**を採用した。静的LCAの結果（FDES/EPDの集計GWP値）に動的係数FRE2020(t)を乗じる方式であり、既存の静的LCAインフラを最大限活用しながら動的補正を追加する実用的アプローチである。木材の正味便益はこの重み付けの非対称性によりIc construction内で発現する（規制上のStockC指標は貯蔵量を開示する別個の情報提供指標）。
3. **No.23（評価結果利用）:** Ic指標の開示は建築確認申請の必須書類となり、設計段階の意思決定、金融機関のESG評価、保険料算定、EUタクソノミー適合判定に活用されつつある。日本でもこのエコシステム全体を参照することが必要である。

## 1. 算定環境（マニュアル、ツール、人材育成）（No.21）

### 1.1 フランスのLCA算定エコシステム三層構造

```
【第1層：データベース】
  INIES (inies.fr)  ← 中央データポータル
  ├── FDES（Fiche de Déclaration Environnementale et Sanitaire）
  │   ・建材メーカーが作成する環境製品宣言（フランス版EPD）
  │   ・2026年現在、約5,500件収録（うち木材系 約800件）
  │   ・バイオジェニック炭素データを含む（一部のみ）
  ├── PEP（Product Environmental Profile）── 電気設備機器
  └── PEP ecopassport ── 建築設備システム
  
  デフォルト値（données par défaut）
  ・FDESが存在しない建材に使用する「保守的（不利側）」標準値
  ・過度に高いGWP値のため、Ic計算を不利にする側面がある（課題）

【第2層：計算ソフトウェア】
  Elodie（CSTB開発・無償提供）─── 主要ツール（シェア80%超）
  ├── バージョン: 2024年版（Elodie v5.x相当）
  ├── 機能: INIESからFDESを直接インポート・Ic計算自動化
  ├── 動的LCA: FRE2020(t)の自動適用・StockC自動算定
  ├── 出力: 申請用PDFレポート（DHUP指定フォーマット）
  └── 動作環境: Windows（Web版も提供中）
  
  その他ツール（CEQUA認証）:
  ├── NOVABASE Edificius LCA── BIM連携特化
  ├── ArchiLCA ── 設計初期段階向け簡易ツール
  └── eTool（豪州）── 一部フランス語版対応

【第3層：申請・認証機関】
  建築確認申請（Permis de construire）  ← 法的義務
  ├── 計算書（Elodie出力PDF）を添付
  ├── 計算書は認証機関の審査（Cerqual, Bureau Veritas等）
  └── BBCA（低炭素建築認証）── 任意認証（Ic=seuil×0.70以下）
```

### 1.2 Elodie計算ツールの詳細仕様

CSTBが開発・無償提供するElodie（エロディ）はRE2020計算の事実上の標準ツールである：

| 仕様 | 内容 |
|------|------|
| **開発機関** | CSTB（Centre Scientifique et Technique du Bâtiment） |
| **提供形態** | 無償提供（Web登録制、フランス居住者・法人向け） |
| **INIES連携** | 5,500件のFDES/PEPを直接インポート可能 |
| **RE2020対応** | FRE2020(t)自動適用・StockC自動算定・Ic_energy連携 |
| **BIM連携** | IFC形式（一部）・Autodesk Revit連携プラグイン（ベータ） |
| **バリデーション** | DHUP認定（法的提出書類の生成が可能） |
| **精度** | 整合化FDES使用時±5%以内（Cerema, 2024 §6.2） |
| **課題** | 複雑建材の動的タイムラインが手入力必須・クラウド連携未成熟 |

### 1.3 人材育成・訓練要件

**資格・訓練体制（2022年〜）：**

| 対象者 | 必要スキル | フランスの対応 |
|--------|-----------|--------------|
| **BET（構造設計事務所）** | Elodie操作・FDESデータ選択・Ic目標設定 | CSTB主催オンライン研修（無償）・動画マニュアル |

| **建材メーカー** | FDES作成・INIESへの登録 | CSTB認定FDES検証機関（有償） |
| **審査機関スタッフ** | 計算書の整合性チェック | DHUPガイド（2022年版） |

**訓練上の課題（Rivaton, 2025 推奨事項 No.3）：**
- 小規模設計事務所でのElodieスキル不足が深刻（中小設計事務所の約40%が外注）
- FDESデータ品質のばらつきを設計者が判断できるリテラシー不足
- 施工段階（A5モジュール）の記録が実務で省略されることが多い

## 2. 静的LCA評価との関係性（No.22）

### 2.1 RE2020のハイブリッド設計思想

RE2020の計算は純粋な動的LCAではなく、**「静的LCAの結果を入力として動的係数を後付けで掛ける」**ハイブリッド方式である：

```
【静的LCAデータ（入力層）】
  FDES/EPD（静的GWP100値。生物由来炭素の固定・放出を含む）
       │
       │ × F_RE2020(t_m)        ← 動的係数を各モジュール排出に適用
       ▼                           （固定 t=0で×1.000、放出 t=50で×0.578）
【動的重み付け後のGWP値（算定層）】
  ─────────────────────────────
  = Ic_construction [kgCO₂eq/m²]  ← RE2020規制指標（出力層）
  ※木材の正味便益は固定と放出の重み差として上記の内部で自動発現
  ※StockC（生物由来炭素貯蔵量 kg C/m²）は別途算出される情報提供指標
```

**このハイブリッド設計の合理性：**
- 既存の静的EPD/FDESインフラを流用 → 新たなデータ整備コストを最小化
- 動的LCI（年次排出プロファイル）の整備を不要とする
- Elodie計算ツールへの実装が容易（係数の掛け算のみ追加）

### 2.2 モジュール別の静的・動的評価マッピング

| LCAモジュール | 内容 | 静的/動的 | $F_{RE2020}(t)$ |
|------------|------|---------|----------------|
| **A1** | 原材料採取 | **静的GWP100** | 1.000（t=0） |
| **A2** | 輸送（製造地→建設現場） | 静的 | 1.000 |
| **A3** | 製造プロセス | 静的 | 1.000 |
| **A4** | 建設現場への輸送 | 静的 | 1.000 |
| **A5** | 施工・建設 | 静的 | 1.000 |
| **B1** | 使用 | 静的 | 0.790（t=25年平均） |
| **B2-B5** | 維持管理・修繕・改修 | 静的 | 0.790（平均） |
| **B6** | 運用エネルギー消費 | 静的 | $Ic_{énergie}$として別計上 |
| **B7** | 運用水消費 | 除外 | — |
| **C1** | 解体 | 静的 | 0.578（t=50年） |
| **C2** | 廃材輸送 | 静的 | 0.578 |
| **C3** | 廃棄物処理 | 静的 | 0.578 |
| **C4** | 廃棄 | **生物由来炭素の放出時点** | 0.578 |
| **D** | 再利用・回収ポテンシャル | 静的 | **Ic除外（任意申告）** |

**バイオジェニック炭素は特別扱い：**
- A1-A3（t=0）: 固定量 $-B$ を $F=1.000$ で計上 → $-B$
- C4（t=50年）: 放出量 $+B$ を $F=0.578$ で計上 → $+0.578B$
- **正味便益 = $-B(1-0.578) = -0.422B$**（負値＝Ic constructionを下げる効果。Bc≈640なら約−270 kgCO₂eq/m³）
- ※これは規制上のStockC指標（貯蔵量 kg C/m²）とは別概念

### 2.3 静的LCAとの差異が大きいケース・小さいケース

**差異が大きいケース（動的補正の効果が顕著）：**
1. バイオジェニック炭素量が多い建材（CLT、LVL、木質パネル等）
2. 使用段階（B段階）で大量の取替え・修繕が発生する建材
3. 解体段階（C段階）で焼却される建材

**差異が小さいケース（動的補正がほぼ効かない）：**
1. バイオジェニック炭素がない建材（コンクリート、鋼材、ガラス）
2. A1-A3のみで評価が完結する短寿命部材
3. 製造段階（A）の化石燃料由来排出が支配的な建材

### 2.4 EN 15804+A2との整合と矛盾

| 規格/規制 | バイオジェニック炭素の扱い | 一時貯蔵評価 | 時間軸評価 |
|---------|----------------------|-----------|-----------|
| **EN 15804+A2（欧州標準）** | −1/+1（A1で−1、C4で+1。正味ゼロ） | なし | なし |
| **ISO 14067** | 開示義務あり（正味ゼロの立場） | なし | なし |
| **EN 15978（建物LCA）** | EN 15804準拠 | なし | なし |
| **RE2020（フランス国内）** | FRE2020(t)適用・StockCクレジット | **あり** | **あり** |

> **矛盾の政治的重要性:** EN 15804は欧州委員会策定の統一規格であり、フランスのRE2020はこれと相容れない。CERIB（コンクリート産業研究所）はこの矛盾を「EU規制との非整合性」として批判しており、欧州内での方法論論争が続いている（RE2020資料 p.8-10）。フランス政府は「EN 15804+A2は科学的進歩に遅れており、改訂が必要」という立場で対抗している。

## 3. 評価結果の利用方法・表示方法（No.23）

### 3.1 Ic指標の法的要件と開示フロー

```
【設計段階（着工前）】
  ① 設計者がElodieでIc試算 → 設計段階での材料比較・最適化
  ② Ic目標値（seuil値）を満たすかチェック
  
【建築確認申請（Permis de construire）】
  ③ Elodie出力のPDFレポートを申請書類に添付（法的義務）
  ④ Ic_construction + Ic_énergie の両方の開示が必要
  
【施工段階】
  ⑤ 設計段階LCA → 実施段階LCAへの更新（A5実績値の反映）
  
【完成・引渡し段階】
  ⑥ 最終Ic値の確定・認証機関への提出
  ⑦ DHUP指定フォーマットで記録保存（10年間）
```

### 3.3 任意認証・ラベル体系との関係

RE2020の義務的Ic開示に加え、より低炭素な建物を認証する任意ラベル体系が存在する：

| ラベル | 運営機関 | Ic要件 | 特徴 |
|--------|---------|--------|------|
| **BBCA（低炭素建築認証）** | BBCA協会 | seuil × 0.70以下 | 最もプレスティッジ。木造建築で取得しやすい |
| **E+C-ラベル** | DHUP（旧実証プログラムの認証継続） | seuil × 0.85以下 | E+C-の成果を認証継続 |
| **NF HQE Bâtiment** | Qualitel | Ic基準含む総合評価 | 高コスト・大規模建物向け |
| **BREEAM（英）** | BRE | 部分的Ic評価 | フランスでは少数派 |

### 3.4 金融・保険分野での活用

Ic指標は建築分野を超えて金融・保険分野でも活用が進んでいる：

| 用途 | 活用方法 | 段階 |
|------|---------|------|
| **グリーンボンド適格性** | Ic値がEUタクソノミー技術基準の参照指標（建物の「実質的貢献」判定） | 検討中（2026年確定予定） |
| **住宅ローン金利優遇** | BBCA認証建物への住宅ローン金利割引（一部銀行） | 実施中（Crédit Agricole等） |
| **保険料算定** | 低Ic建物の長寿命性を前提にした保険モデル | 試験的 |
| **ESG報告** | 不動産ポートフォリオのScope 3排出量計算にIc値を使用 | 実施中（大手不動産ファンド） |

### 3.6 国際的比較：他国の建物LCA開示制度

| 国/地域 | 制度名 | 動的LCA | Ic相当指標 | 法的義務 |
|--------|-------|--------|----------|---------|

### 3.7 産業界の反応と活用状況

**推進側（木材・CLT建材業界）の活用：**
- CLT製品のFDES登録を積極化（StockCクレジットの最大化）
- Ic値の低さを「マーケティング指標」として活用（BBCA認証取得建物の広告）
- 設計者向けの「材料比較シミュレーター」提供（FCBA開発）

**抵抗側（コンクリート・鋼材業界）の対応：**
- CERIB：「EU規格と整合しない国内規制は将来改訂されるべき」と継続的主張
- 鋼材産業：「Dモジュール（リサイクル鋼材の炭素便益）をIcに組み込むべき」と要求
- 双方の主張は2028年フェーズ3改訂の議題として正式に取り上げられている
